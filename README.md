# Abschlussprojekt_Brauerei


# Projekt initialisieren
```
npm create vite braurei -- --template react
```
## Styling
```
npm install react-bootstrap bootstrap
```
```
App.jsx

import 'bootstrap/dist/css/bootstrap.min.css';
```

# Projekt staten
```
npm run dev
```
